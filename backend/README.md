# Backend

This folder contains the initial Express.js backend for the Smart Hydroponics Monitoring Dashboard.

## Files

- `server.js` - Starts the application and listens on the configured port.
- `src/app.js` - Creates the Express app and defines the base middleware and health check response.
- `.env.example` - Documents the environment variables needed for local setup.
- `.gitignore` - Excludes generated, local, and sensitive files from version control.
- `README.md` - Describes the purpose of the backend starter files and setup direction.

## Notes

This starter intentionally avoids adding routes, controllers, or models at this stage.
